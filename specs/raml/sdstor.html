/namespaces

/namespaces/{nsid} get

get

Get detail view about namespace

get /namespaces/{nsid}

Get detail view about namespace

Secured by itsyouonline with scopes:
  • user:memberof:store.namespace

OAuth2 is a protocol that lets external apps request authorization to private details in a user's account without getting their password.

  • Request
  • Response
  • Security

URI Parameters

  • nsid: required (string)

HTTP status code 200

Body

Media type: application/json

Type: object

Properties
  • label: required (string - minLength: 5 - maxLength: 128 - pattern: ^\w+$)
  • stats: (object)

    statistics of the namespace

    • spaceUsed: (number)

      Total size in MiB used in the namespace

    • spaceAvailable: (number)

      Size in MiB of remaining in the namespace. It the result of (totalSize - spaceUsed)

    • readRequestPerHour: required (integer)

      Number of read request per hour

    • writeRequestPerHour: required (integer)

      Number of write request per hour

    • nrObjects: required (integer)

      Number of object contained in this namespace

Example:

properties:

{
  "label": "MyLabel",
  "stats": {
    "spaceAvailable": 128,
    "spaceUsed": 10,
    "nrObjects": 400,
    "readRequestPerHour": 250,
    "writeRequestPerHour": 120
  }
}
